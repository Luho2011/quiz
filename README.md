#  Interactive Quiz App

Interactive quiz application with multiple game modes, focusing on dynamic user interactions and UI mechanics.

## 🚀 Live Demo
👉 https://celeb-quiz.vercel.app/

## 🛠️ Tech Stack
- React
- CSS
- Vercel

## ✨ Features
- Multiple little quiz games with different gameplay mechanics  
- Drag & drop interactions  
- Dynamic UI behavior and state-driven logic  
- Integration of the Spotify API (Hitster)

## ⚙️ Challenges & Decisions

1. Quiz State Management
   
   Managing quiz state, user interactions and tracking quiz progress.

2. Score Calculation
   
   Dynamically calculating and displaying scores in real-time based on user inputs.

3. External API Integration & Drag & Drop
   
   Integrating the Spotify API to implement the "Hitster" game (Drag & Drop), fetching song titles, artists, and release dates.
